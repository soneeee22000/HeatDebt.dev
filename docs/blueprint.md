# **App Name**: HeatAlert Dashboard

## Core Features:

- Interactive District Map: Display an interactive map of Montgomery districts, visually outlining each area with clickable regions.
- Heat Risk Overlay: Visualize district-specific 'heat risk' levels directly on the map using a color-coded heatmap with an accompanying legend.
- District Detail Panel: Dynamically load and display a detailed view of key data (current heat index, neighborhood vitals, community facilities) for a selected district in a responsive side panel.
- Live Data Indicator: Implement a visual 'Live Sensor Feed' indicator, suggesting real-time or frequently updated data display (mock data for MVP).
- Comprehensive Data Display: Present a range of neighborhood vital statistics such as population, green space, pollution rate, and AC access within the detail panel.
- AI Grant Report Tool: Generate a summarized narrative report suitable for grant applications by leveraging district-specific data and identified needs through a generative AI tool.

## Style Guidelines:

- A deep, slightly desaturated crimson background (#251A1D) to provide a rich, dark canvas, allowing data visualizations to stand out.
- Primary interactive elements and alert highlights use a vibrant, energetic red-pink (#E83D60), symbolizing critical data and drawing attention to actionable insights.
- Accent elements and secondary interactive states utilize a rich, deep magenta-purple (#AA2182) for modern sophistication and contrast, complementing the primary red without competing.
- All text uses 'Inter' (sans-serif) for both headlines and body, ensuring readability and a clean, modern aesthetic fitting for data display. Numbers and key metrics will be bold for emphasis.
- Minimalist, crisp line icons are used throughout the interface, with selected states featuring subtle glow effects to highlight interaction or live data.
- A clear two-column layout with an immersive map on the left and a dynamic data details panel on the right, enabling easy navigation and information retrieval on a single page.
- Smooth, subtle transitions for district selection and data panel updates, alongside a gentle pulsing or glowing animation for the 'Live Sensor Feed' indicator.
