## Challenge Categories

Pick one of four civic challenge streams:

1. **Civic Access** — Resident-facing tools, service finders, chatbots
2. **Workforce Growth** — Jobs, skills, economic development tools
3. **Smart Cities** — Data dashboards, infrastructure analytics
4. **Public Safety & Emergency Response** — Safety, crime, emergency tools

All challenges are centered on the **City of Montgomery, Alabama** and its open data.

---

## What You're Building

A working AI-powered prototype that:
- Solves a real civic problem using Montgomery's open data
- Uses AI/ML in a meaningful way (not just a wrapper)
- Is demoed via a short video + live URL

---

## Judging Criteria

Projects are scored on five dimensions:

| Criterion | What Judges Look For |
|---|---|
| **Relevance** | Directly addresses a real civic challenge using Montgomery data |
| **Execution Quality** | Working prototype, clean demo, no broken features |
| **Originality** | Novel approach — not a generic dashboard |
| **Social Impact** | Measurable benefit to Montgomery's 200,000+ residents |
| **Commercial Potential** | Could the city adopt it? Could it scale to other cities? |

**Bonus credit** for excellence in vibe coding (AI-assisted prototyping).

## Data & Tools

### Primary Data Source
**City of Montgomery Open Data Portal**
https://opendata.montgomeryal.gov
- Powered by ArcGIS Hub with REST API access
- Datasets: crime stats, building permits, business licenses, code violations, parks, recreation, 311 requests, and more
- Free, public, no authentication required for most datasets

## Rules Summary

- Participation is free and open to anyone worldwide
- No prior coding experience required
- Teams of 1–5 members
- Must build something new for this hackathon (no pre-built projects)
- Open-source tools, APIs, and AI models are allowed
- Only the Team Representative (captain) needs to submit
- Late submissions accepted with a penalty: **1 point deducted per 2-hour window** past deadline, max 5 points deducted
- Completing and submitting earns an official GenAI Works certificate
